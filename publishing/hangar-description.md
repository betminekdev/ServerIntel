# ServerIntel

Smart staff assistant for Minecraft servers.

ServerIntel helps staff teams understand suspicious player behavior using risk scores, player timelines, smart alerts, and watch mode.

## Honest Positioning

ServerIntel is not a classic anti-cheat. It does not auto-ban players, does not inspect clients, and does not claim perfect xray detection.

It provides server-side signals and investigation data so staff can review behavior faster.

## Included in v0.1.0-beta

- Player risk score
- Risk levels
- Player timeline
- Suspicious mining signals
- High-value ore burst detection
- Staff alerts
- Watch mode
- SQLite storage
- Configurable thresholds

## Commands

Use `/si` for daily staff work.

The full command is `/serverintel`. Aliases `/smartadmin` and `/sa` are also registered.

## Beta Notice

This is a beta release. Test before production use and tune the configuration for your server rules and staff workflow.
