# Security Policy

SmartAdmin is a Minecraft server-side staff assistant. It stores investigation data such as player profiles, risk scores, and timeline events.

## Reporting Security Issues

Please report security or abuse-related issues responsibly.

For private reports, contact the maintainer through GitHub or another private channel listed by the project owner. Do not publish exploit details publicly until the maintainer has had time to review and respond.

Useful details include:

- SmartAdmin version
- Paper/Spigot version
- Java version
- Description of the issue
- Steps to reproduce
- Relevant logs
- Whether private player data or server data could be exposed

## Abuse Reports

SmartAdmin does not auto-ban players and does not claim guaranteed cheat detection. If a configuration or feature could encourage unsafe moderation decisions, please report it as an abuse-risk issue.

## Supported Versions

During the beta period, only the latest public beta is expected to receive fixes.
